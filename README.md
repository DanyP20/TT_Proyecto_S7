# TT_Proyecto_S7
Dashboard interactivo de proyecto Sprit Ds

En este proyecto pudimos desarrollar una app mediante la terminal de la computadora y la de Visual Code Studio, utilizando Git y GitHub, creando un repositorio con lo necesario para diseñar una app muy basica, consiste en crear un panel de control interactivo que permite visualizar y explorar datos de venta de coches sin necesidad de programar. Al desarrollarlo, obtienes experiencia práctica construyendo una aplicación web real, similar a lo que hace un analista de datos o un ingeniero de software en el mundo laboral. 

Además, se convierte en una parte valiosa de tu portafolio profesional, ya que demuestra tus habilidades técnicas y tu capacidad para crear soluciones completas. También funciona como una herramienta de análisis con gráficos, histogramas y opciones de filtrado, lo que facilita que cualquier persona, incluso sin conocimientos técnicos, pueda interpretar y analizar la información.
